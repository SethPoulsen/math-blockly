Maths Blocks: A visual mathematical expression builder

Maths Blocks is a system for constructing and manipulating mathematical expressions from visual blocks representing syntactical elements.

Some demonstrations:

  • Builder - freely build expressions
  • Definition of limit, convergence and divergence - example of an online tutorial/quiz question
  • Manipulation of logic symbols - example of online tutorial activities
  • Translation from English to mathematical notation - example of online practice activities
  • Vector and scalar arithmetic - example of abstract operations using polymorphic blocks
Maths Blocks is based on the Blockly project. Source is available at Github. Gratefully using code and inspiration from Polymorphic Blocks, App Inventor and HenrikD's Blockly plugins.
